### 推箱子游戏  

#### 适用平台

J2ME-280*320  

#### 开发平台

Ubuntu12.04LTS-i386 + eclipse-Indigo + eclipseme.feature_1.7.9 + sun_java_wireless_toolkit-2_5_2（选用设备defaultColorPhone）  

#### 游戏简介

在一个狭小的仓库中，有一些散乱放置的箱子，BOSS要求你将所有箱子摆放到合适的位置。你能做到吗？
    
使用方向键可以控制玩家移动，但你和箱子都无法穿过墙壁，你也不能同时推动两个箱子哦。
    
为了这个月的奖金，赶快开始吧！
